# Model-Free Vertical Text OCR

> **Domain:** Logistics

## Overview

Training-free solution for reading vertically-oriented shipping labels using image preprocessing, morphological operations, and EMNIST character matching without custom model training.

## Skills & Technologies

- OpenCV
- Image Preprocessing
- EMNIST Matching
- Tesseract OCR
- Bounding Box Clustering
